# Work with arrays in elegant way

## For contributors 

**Pass all tests locally before create pull request.**

Build container
```shell
composer docker-build
```
Install dependencies
```shell
composer docker-install
```

All tests
```shell
composer check-all
```

Run tests
```shell
composer unit
```

Ensure coverage is 100%
```shell
composer coverage
```

Test code style
```shell
composer style
```
